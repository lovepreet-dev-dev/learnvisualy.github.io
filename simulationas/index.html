Learning Design

Each lab follows the same loop

1. Manipulate inputs

Adjust sample sets, priors, loss parameters, cluster counts, observation sequences, or neighborhood radii before the algorithm runs.

2. Advance one step

Use the step controls to reveal sorting, assignment, message passing, split selection, parameter updates, or posterior inference one action at a time.

3. Read the logic

The explanation panels translate every update into plain language: what changed, why it changed, and what would happen if you changed the input again.
