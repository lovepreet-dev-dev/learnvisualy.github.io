# Machine Learning Studio

Interactive static web pages for learning core machine learning algorithms with step-by-step explanations.

## Open the site

Open [`index.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/index.html) directly in a browser, or serve the folder with any static file server.

## Included pages

- [`index.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/index.html): curriculum overview
- [`pages/introduction.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/pages/introduction.html): estimation, MLE, MAP, Bayesian estimation, bias-variance, KDE, and noisy data cleanup
- [`pages/classification.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/pages/classification.html): k-NN, decision trees, CART, LDA, logistic regression, perceptrons, and SVM-style large-margin learning
- [`pages/graphical-models.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/pages/graphical-models.html): Bayesian networks, MRFs, belief propagation, Markov chains, and HMMs
- [`pages/clustering.html`](/Users/lovepreet/Desktop/ vibe projects/simulationas/pages/clustering.html): K-means, K-medoids, hierarchical clustering, DBSCAN, and spectral clustering

## Notes

- All interactions run in plain JavaScript with no build step.
- The site is designed for local exploration and teaching rather than production deployment.
